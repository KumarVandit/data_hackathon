# Reference: Graphiti MCP Server

This directory contains the reference implementation of the Graphiti MCP Server from [getzep/graphiti](https://github.com/getzep/graphiti/tree/main/mcp_server).

## Purpose

These files are kept for reference to understand the Graphiti MCP Server architecture and implementation patterns. They are **not** used directly in the project - our implementation is in `../mcp_server/`.

## Contents

- `main.py` - Main entry point for the Graphiti MCP server
- `src/` - Source code modules
- `config/` - Configuration files
- `docker/` - Docker setup files
- `tests/` - Test files
- `README.md` - Original Graphiti documentation
- `pyproject.toml` - Python project configuration
- `uv.lock` - Dependency lock file

## Usage

Refer to these files when:
- Understanding MCP server patterns
- Learning Graphiti architecture
- Comparing implementation approaches
- Finding best practices

## Note

Do not modify these reference files. They are kept as-is from the original repository for reference purposes only.
